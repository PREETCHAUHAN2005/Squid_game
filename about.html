About This GFG Project

Commitment to Clean Code and Responsive Design.

Our Motivation: The Club Application

This project is our application to the GeeksforGeeks club. We chose the **Squid Game theme** to showcase our ability to create a visually distinct, high-quality, and modern web application using **Pure HTML5, CSS3, and Vanilla JavaScript**. We focused on delivering a professional, responsive, and performant user experience.

Project Requirements Fulfilled

The design strictly adheres to the requested constraints:

  • **Theme:** Authentic Squid Game visual elements, colors (Pink, Green, Black).
  • **Framework:** Pure HTML, CSS, and JS (no frameworks/libraries used).
  • **Design:** Visually striking, fully responsive layout for mobile and desktop.
  • **Code Quality:** Clean, well-structured, and easily readable semantic HTML.
  • **Interactivity:** A functional countdown timer built with Vanilla JavaScript.

Design Philosophy

We used CSS custom properties (variables) for the color scheme to ensure consistent theming. The layout relies on modern CSS features like Flexbox and Grid for robust responsiveness, ensuring cross-browser compatibility and quick load times.
